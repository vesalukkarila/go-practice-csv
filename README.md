# go-practice-csv

Pair programming exercise: Repository for practicing csv parsing and other stuff.

## Usage

### Run locally

```sh
go run main.go
```

### Run tests

```sh
go test ./...
```

### Build and run binary 

1. First build it:
    ```sh
    go build -o main
    ```

2. Run the binary:
    ```sh
    ./main 
    ```
